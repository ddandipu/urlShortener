# TinyApp Project

a full stack web app built with node and express. Allows users to shorten longer
web site address.


# dependancies

- Node.js
- Express
_ EJS
-bcrypt
- body-parser
- cooke-session

## Getting started

- Install all dependencies, use npm install
  command.

- Run the development web server using the "node express_server.js command"
